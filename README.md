# 💰 ERP - Sistema de Gestão de Caixa

Sistema ERP desenvolvido para controle financeiro empresarial, permitindo o gerenciamento de vendas, despesas, compras e
relatórios financeiros.

O objetivo do projeto é automatizar processos administrativos e oferecer melhor organização do fluxo de caixa de pequenas empresas.

---

# 🚀 Tecnologias Utilizadas

## Backend
- Python
- Flask
- SQLite

## Frontend
- React Native
- TailwindCSS

## Ferramentas
- Git
- GitHub
- Node.js

---

# 📊 Funcionalidades

- Cadastro de vendas
- Controle de despesas
- Registro de compras
- Dashboard financeiro
- Relatórios de movimentação
- Integração entre front-end e API
- Operações CRUD completas

---

# 🧠 Objetivo do Projeto

Este projeto foi desenvolvido com foco em aprendizado prático de desenvolvimento full stack, aplicando conceitos como:

- construção de APIs REST
- modelagem de banco de dados
- integração entre front-end e back-end
- organização de código e versionamento
